### Colaboradores
1. Clonar el repositorio 
2. Entrar a la carpeta 
```bash
cd gitworkflow2/
```
3. Antes de subir la modificacion hacer 
```bash
git pull
```
4. Despues subir nuestros cambios 
```bash
git add . 
git commit -m ""
```
5. Despues subir lo que realizamos 
```bash
git push
```
